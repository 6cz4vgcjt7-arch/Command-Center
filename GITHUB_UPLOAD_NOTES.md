# GitHub Upload Notes

Upload the files in this ZIP directly to the root of the existing `Command-Center` repository.

Recommended:
1. Open the repository on GitHub.
2. Click **Add file** → **Upload files**.
3. Drag the extracted contents of this ZIP.
4. Commit directly to `main`.

Commit message:
`Seasons v1.2.0 foundation freeze`

After upload:
- Open Settings in the app and confirm version shows `v1.2.0 / Build 120`.
- If the old version still appears, close/reopen the home-screen app or refresh Safari.
