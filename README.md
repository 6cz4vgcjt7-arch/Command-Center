# Seasons v0.3

A local-first weekly financial review prototype.

## What's included

- Command, Review, Accounts, and Settings tabs
- Local browser storage only
- Account-by-account Weekly Review
- Four-segment review cycle
- Week Closed state
- Editable review day, time, and focus strategy
- Promotional APR tracking with expiration date and weekly reviews remaining
- Export backup and reset local data

No bank connections. No personal data included.
