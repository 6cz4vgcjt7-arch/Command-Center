# Command Center — GitHub Pages Deploy

## Fast deploy steps
1. Create a GitHub repository named `command-center`.
2. Upload every file in this folder to the repository root.
3. Go to Settings → Pages.
4. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Open the GitHub Pages URL on iPhone Safari.
7. Tap Share → Add to Home Screen.

## Notes
- Data is stored locally in the browser.
- No bank connection.
- No AI.
- No dollar amounts on the Home Screen except inside Accounts.
