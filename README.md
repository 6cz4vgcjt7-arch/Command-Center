# Seasons v1.1.0 Founder Update

Seasons is a weekly financial habit app that helps people build lasting financial achievements through intentional observation, reflection, and steady progress.

## New in v1.1.0

- Foundation accounts: Emergency Fund and Retirement.
- Debt and Foundation accounts are interpreted differently during Weekly Review.
- Season Check-In flow.
- Season Transition reflection and encouragement.
- Updated product documents.

## Core Philosophy

The goal isn't to stay in one season forever. The goal is to recognize what deserves your attention today.

## Install

Upload the ZIP contents to the existing GitHub Pages repository and commit directly to `main`.
